# Project template

Copy this directory to `shared_context/projects/<slug>/` when starting a new project. `run-project.js` does this automatically.

The `state.json.template` and `brief.md.template` files are scaffolds — `state-machine.js` initializes the real `state.json` and the orchestrator writes the real `brief.md` from BigBrain's translated brief.

See `protocols/shared-context-schema.md` for the full directory contract.
